WoIstMeinDozent
===============

StudIP Plugin das den Stundenplan eines Dozenten ausgibt.

Das Plugin überprüft nur die Vorlesungen und gibt diese bei Bedarf aus.


Features
====
- Suche eines Dozenten
- Ausgabe aller Vorlesungstermine die er hat
BUGS
====
- Wenn kein Datum gewählt wurde, wird kein Plan ausgegeben
